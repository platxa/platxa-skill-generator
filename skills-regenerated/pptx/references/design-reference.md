# Design Reference for Presentations

Color palettes, visual patterns, and layout guidance for creating professional presentations.

## Color Palette Selection

Pick 3-5 colors that work together: dominant + supporting + accent. Think beyond defaults -- consider topic, industry, mood, energy, audience, and brand identity.

## Layout Patterns

**Charts/Tables**: Use two-column layout (text + visual) or full-slide. Never vertically stack text above chart.

**Geometric**: Diagonal dividers, asymmetric columns (30/70, 40/60), rotated headers, overlapping shapes for depth.

**Borders**: Thick single-side borders, L-shaped frames, underline accents beneath headers (3-5pt).

**Typography**: Extreme size contrast (72pt headlines vs 11pt body), all-caps with wide spacing, monospace for data, condensed fonts for dense info.

**Backgrounds**: Solid color blocks (40-60% of slide), split backgrounds, edge-to-edge bands, gradient fills (pre-rendered as PNG).

**Data Styling**: Monochrome charts with single accent, horizontal bars, minimal gridlines, oversized key metrics, data labels on elements (no legends for single series).
