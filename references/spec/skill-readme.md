# Skill README Documentation

Guidelines for creating README.md files that explain skill installation and usage.

## README Purpose

The README.md file serves as external documentation for skill distribution:
- Installation instructions
- Usage examples
- Configuration options
- Troubleshooting guide

**Note:** README.md is optional and separate from SKILL.md. SKILL.md is the skill itself; README.md explains the skill to users.

## README Structure

```markdown
# Skill Name

Brief description of what the skill does.

## Installation

### User Installation (Global)

```bash
cp -r skill-name ~/.claude/skills/
```

### Project Installation (Local)

```bash
cp -r skill-name ./.claude/skills/
```

## Usage

```
/skill-name
```

### Options

| Option | Description |
|--------|-------------|
| `--option1` | Description |
| `--option2` | Description |

## Examples

### Example 1: Basic Usage

```
/skill-name target
```

### Example 2: With Options

```
/skill-name --option1 value target
```

## Configuration

Configuration options and customization.

## Troubleshooting

Common issues and solutions.

## License

License information.
```

## Troubleshooting

### Skill Not Found

If `/{{skill-name}}` doesn't work:

1. Verify installation location:
   ```bash
   ls ~/.claude/skills/{{skill-name}}/SKILL.md
   ```

2. Check file permissions:
   ```bash
   chmod -R 644 ~/.claude/skills/{{skill-name}}/*.md
   ```

3. Restart Claude Code

### Unexpected Behavior

1. Check SKILL.md for syntax errors
2. Verify YAML frontmatter is valid
3. Review references for broken links

## README Generator

```python
from dataclasses import dataclass
from pathlib import Path

@dataclass
class ReadmeConfig:
    skill_name: str
    skill_title: str
    description: str
    features: list[str]
    options: list[dict]
    examples: list[dict]
    settings: list[dict]
    has_references: bool
    references: list[str]
    has_scripts: bool
    scripts: list[str]
    license: str
    author: str

class ReadmeGenerator:
    """Generate README.md for skills."""

    TEMPLATE_PATH = "templates/readme-template.md"

    def generate(self, config: ReadmeConfig, skill_path: Path) -> str:
        """Generate README content."""
        lines = []

        # Title and description
        lines.append(f"# {config.skill_title}")
        lines.append("")
        lines.append(config.description)
        lines.append("")

        # Features
        lines.append("## Features")
        lines.append("")
        for feature in config.features:
            lines.append(f"- {feature}")
        lines.append("")

        # Installation
        lines.append("## Installation")
        lines.append("")
        lines.append("```bash")
        lines.append(f"cp -r {config.skill_name} ~/.claude/skills/")
        lines.append("```")
        lines.append("")

        # Usage
        lines.append("## Usage")
        lines.append("")
        lines.append("```")
        lines.append(f"/{config.skill_name}")
        lines.append("```")
        lines.append("")

        # Options
        if config.options:
            lines.append("### Options")
            lines.append("")
            lines.append("| Option | Description |")
            lines.append("|--------|-------------|")
            for opt in config.options:
                lines.append(f"| `--{opt['name']}` | {opt['description']} |")
            lines.append("")

        # Examples
        lines.append("## Examples")
        lines.append("")
        for i, ex in enumerate(config.examples, 1):
            lines.append(f"### Example {i}: {ex['title']}")
            lines.append("")
            lines.append("```")
            lines.append(ex['command'])
            lines.append("```")
            lines.append("")

        # File structure
        lines.append("## File Structure")
        lines.append("")
        lines.append("```")
        lines.append(f"{config.skill_name}/")
        lines.append("├── SKILL.md")
        lines.append("├── README.md")
        if config.has_references:
            lines.append("├── references/")
            for ref in config.references:
                lines.append(f"│   └── {ref}")
        if config.has_scripts:
            lines.append("└── scripts/")
            for script in config.scripts:
                lines.append(f"    └── {script}")
        lines.append("```")
        lines.append("")

        # License
        lines.append("## License")
        lines.append("")
        lines.append(config.license)
        lines.append("")

        return "\n".join(lines)

    def write(self, config: ReadmeConfig, skill_path: Path) -> Path:
        """Write README.md to skill directory."""
        content = self.generate(config, skill_path)
        readme_path = skill_path / "README.md"
        readme_path.write_text(content)
        return readme_path
```

## When to Include README

| Scenario | Include README |
|----------|----------------|
| Personal skill | Optional |
| Shared with team | Recommended |
| Published publicly | Required |
| Complex configuration | Recommended |
| Multiple scripts | Recommended |

## README vs SKILL.md

| Aspect | SKILL.md | README.md |
|--------|----------|-----------|
| Purpose | Skill instructions for Claude | Human documentation |
| Audience | AI | Developers |
| Required | Yes | No |
| Format | YAML frontmatter + Markdown | Standard Markdown |
| Content | How to execute the skill | How to install/configure |
